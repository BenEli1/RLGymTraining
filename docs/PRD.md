# Product Requirements Document

Build a professional Exercise 3 submission that demonstrates REINFORCE and A2C for sequential workout recommendation over an LSTM world model.

Success means the repository has runnable code, tests, configuration, documentation, and honest experiment reporting. The user should be able to prepare data, train the LSTM, train REINFORCE, train A2C, evaluate against a random baseline, and inspect outputs.

Non-goals: medical advice, production safety guarantees, GUI, or claiming real Kaggle results without the dataset.
